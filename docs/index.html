Getting Started

A simple documentation generator for your project.

Easily build static HTML pages from your markdown files.

Installation

1go install github.com/Nigel2392/doccer@latest

Usage

1doccer init  # Initialize a new skeleton for the documentation.
2doccer serve # Serve the documentation with a local server.
3doccer build # Build the documentation.
